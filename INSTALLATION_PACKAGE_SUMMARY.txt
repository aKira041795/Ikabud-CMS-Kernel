================================================================================
SYSTEM REQUIREMENTS
================================================================================

MINIMUM:
- OS: Linux (Ubuntu 20.04+, Debian 10+, CentOS 8+)
- PHP: 8.1+
- Database: MySQL 8.0+ or MariaDB 10.5+
- Web Server: Apache 2.4+ or Nginx 1.18+
- Memory: 2GB RAM
- Disk: 2GB free space

RECOMMENDED (Production):
- CPU: 4+ cores @ 3.0 GHz
- RAM: 8+ GB
- Disk: 50+ GB SSD
- Network: 1 Gbps

See REQUIREMENTS.md for complete details.
